1： ajax 是什么？有什么作用？

AJAX即“Asynchronous JavaScript and XML”（异步的JavaScript与XML技术），使用浏览器提供的XMLHttpRequest对象，利用这个对象使得浏览器可以发出HTTP请求与接收HTTP响应。无需加载整个页面就能更新用户界的技术。

作用：网页与服务器交换数据，实现网页的异步加载，局部更新网页，提高用户体验和减少服务器带宽。

2： 前后端开发联调需要注意哪些事情？后端接口完成前如何 mock 数据？

前后端联调注意事项：

  • 约定数据：需要哪些数据，数据的类型是什么；响应的数据格式；
  • 约定接口：确定接口名称；请求的方法；请求的参数；
  • 根据这些约定整理接口文档

如何mock数据：

  1. 在本地搭建一个服务器（如：npm install -g server-mock）
  2. 根据文档接口，使用伪数据验证页面的响应和接口是否正常

3：点击按钮，使用 ajax 获取数据，如何在数据到来之前防止重复点击?

设置状态锁，利用if判断语句，判断状态锁的状态。例如：状态锁开始设置为true，发送请求之后变为false，如果响应数据没有到来则return，不再响应事件程序；响应数据到来之后再设置为true。即可防止重复点击。
