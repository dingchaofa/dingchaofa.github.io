1： jQuery 中， $(document).ready()是什么意思？

当某些js代码需要放在body的前面时，为了避免js阻塞后续资源加载或者执行某个事件时无效。就需要把放在body前面的js放在$(document).ready(handle(){navtive code})内部；意即，在文档节点加载完之后，指定一个函数来执行。

window.onload()是文档的所有资源加载完才会执行，而.read()是文档节点加载完就执行。而且里面的变量属于局部变量。

2：$node.html()和$node.text()的区别?

node.html()是获取节点的内容包括节点本身。

node.text()只是获取当前节点及子节点的文本内容，不包括html标签。

5： jQuery 中 data 函数的作用

在匹配元素上存储任意相关数据 或 返回匹配的元素集合中的第一个元素的给定名称的数据存储的值。.data() 方法允许我们在DOM元素上绑定任意类型的数据,避免了循环引用的内存泄漏风险。

.data( key, value ) .data( obj )在匹配元素上存储任意相关数据.

.data( key )返回匹配的元素集合中的第一个元素的给定名称的数据存储的值。 通过.data(name, value)或HTML5 data-* 属性设置

6：写出以下功能对应的 jQuery 方法：

  • 给元素 $node 添加 class active，给元素 $noed 删除 class active

    $node.addClass(‘active’)

    $node.removeClass(‘active’)

    $node.removeClass(‘hover’).addClass(‘active’) //先移除，再增加

  • 展示元素$node, 隐藏元素$node

    展示：$node.show() ; $node.fadeIn(); $node.slideDown();$node.animate(propertier) ;

    隐藏：$node.hide(); $node.fadeOut(); $node.slideup() $node.animate(propertier)

  • 获取元素$node 的 属性: id、src、title， 修改以上属性

    获取：$node.attr('id') $node.attr('src') $node.attr('title')

    修改：$node.attr('id','idvalue') $node.attr('src','srcvalue') $node.attr('title','titlevalue')

  • 给$node 添加自定义属性data-src

    $node.attr('data-src','data-srcvalue')

    如果属性存在就修改，不存在就创建

    当设置样式名（“class”）属性时，必须使用引号！

  • 在$ct 内部最开头添加元素$node

    $ct.prepend($node)

  • 在$ct 内部最末尾添加元素$node

    $ct.append($node)

  • 删除$node

    $node.remove()// 数据无法恢复

    $node.detach() //.detach()保存所有jQuery数据和被移走的元素相关联。当需要移走一个元素，不久又将该元素插入DOM时，这种方法很有用。

    .remove() 将元素移出DOM。 当我们想将元素自身移除时我们用 .remove()，同时也会移除元素内部的一切，包括绑定的事件及与该元素相关的jQuery数据。要删除的元素不删除数据和事件的情况下，使用.detach()来代替。

  • 把$ct里内容清空

    $ct.empty() //不包括自身，只是清空所有子节点

  • 在$ct 里设置 html

    $ct.html(content) //把$ct里的内容替换为content

  • 获取、设置$node 的宽度、高度(分别不包括内边距、包括内边距、包括边框、包括外边距)

    获取宽高: $node.css(['width','height'])

    设置宽高: .css({ width:value1,height:value2})

    不包括内边距: $node.width() $node.height()

    包括内边距:$node.innerHeight() $node.innerWidth()

    包括边框$node.outHeight() $node.outWidth()

    包括外边距: $node.outHeight(true) $node.outWidth(true)

    在 DOM 标记法中，属性名可以不使用引号包裹，但是在 CSS 标记法中，如果属性中含有连字符(-)的话，则必须用引号包裹。。

    注意.height()总是返回容器的高度,不管CSS box-sizing属性值。截至jQuery 1.8，这可能需要检索的CSS的宽度加加上box-sizing的属性，然后当元素有 box-sizing: border-box时候，减去个元素上任何潜在border和padding值。为了避免这种问题，使用.css( "height" )而非.height()。

    .css('height') 和 .height()之间的区别是后者返回一个没有单位的数值（例如，400），前者是返回带有完整单位的字符串（例如，400px）。当一个元素的高度需要数学计算的时候推荐使用.height() 方法 。

  • 获取窗口滚动条垂直滚动距离

    window.scrollY

    $(window).scrollTop

  • 获取$node 到根节点水平、垂直偏移距离

    $node.offset()

  • 修改$node 的样式，字体颜色设置红色，字体大小设置14px

    $node.css({color: ‘red’ , ‘font-size’: ‘14px’ })

  • 遍历节点，把每个节点里面的文本内容重复一遍

    $node.each(function(){ $(this).clone().appendTo(this) })

  • 从$ct 里查找 class 为 .item的子元素

    $ct.children('.item')

  • 获取$ct 里面的所有孩子

    $ct.find()

  • 对于$node，向上找到 class 为’.ct’的父亲，在从该父亲找到’.panel’的孩子

    $node.parent('.ct').children('panel')

    $node.parent('.ct').find('.panel')

  • 获取选择元素的数量

    $node.length

  • 获取当前元素在兄弟中的排行

    $node.index()
