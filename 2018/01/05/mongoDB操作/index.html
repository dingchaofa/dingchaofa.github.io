mongoDB

Jan 05, 2018数据库点击

本操作是在macOS10.12系统上操作和Ubuntu14.04，mongoDB版本是3.4

1. mongoDB安装启动

具体安装步骤网上有很多教程，这里只说自己遇到的一些坑

  1. 参照官网 命令行 export PATH=<mongodb安装目录>/bin:$PATH写入~/.zshrc
    这样可以方便直接在命令行执行mongo shell
  2. mongod是启动mongoDB服务，mongo是启动mongoDB命令环境
  3. 在我的系统下，操作发现并不能通过mongod来启动服务，需要mongod --dbpath data/db，我并没有把data/db目录放在/目录下，直接放在mongoDB的安装目录下。我还以为我安装的有问题，看文档说是自定义创建目录，是通过mongod –dbpath data/db来启动。
  4. 在ubuntu上启动和macOS上是不一样的，一定要看清文档
    sudo service mongod start

2. mongoDB操作

  1. 创建数据库
    use database_name #如果新创建的数据库里没有任何数据，则不会通过show dbs显示此数据库，use也可以用来切换数据库
  2. 删除数据库
    db.dropDatabase() #默认删除当前数据库
  3. 删除集合
    db.collection.drop() #集合就相当于sql类的table
